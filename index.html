Pac-Man

❮ ❯

A recreation of the classic arcade game, Pac-Man, written in Java. This version of Pac-Man contains almost all of the same elements that the original game does, as well as an additional custom maze. At the beginning of each level, Ghosts travel randomly around the maze. After a set time, the Ghosts become "smart" and begin to converge on Pac-Man's location. As the levels go up, the Ghosts become smart more quickly. Watch out!!

The simple GUI allows for a user to enter a three-character ID so that their high score can be tracked at the end of each game. The threaded application runs at 60 fps and calls the tick() method once per frame. During each call to tick(), the position of Pac-Man is fed to the Ghosts and they calculate their own path to him. The path a ghost takes is not necessarily always the shortest one in an effort to make the game last a little longer.

Use the arrow keys or WASD to control Pac-Man.

Countdown Clock

❮ ❯

This is a Countdown Clock application written in Java. This application was developed for CNBC for use on their video wall during live broadcasts.

The application has both count down and count up features. It can count down to a specific time, count up starting "now" or at a specific time, and create a time shifted countdown. A time shifted countdown is used when a countdown needs to be pre-recorded for use in a later broadcast. The user interface features a variety of ways to change specific features on the clocks. The font, color, numeral size, spacing between numerals, and units displayed can all be adjusted and customized. Clocks can also be saved and then loaded back into the program if the countdown has not yet expired.

The application is frequently used to display countdowns to specific events such as the Opening and Closing Bells of a day's trading session or a countdown to Christmas.


*This was a work for hire contract with CNBC. As a result, I do not own the source code and cannot post it.

Pente

❮ ❯

This program is a recreation of the classic strategy game, Pente. It was developed entirely in Java. The application contains a simple GUI for playing the game as well as full documentation for the rules of the game. The GUI has a status bar at the bottom that displays when a player scores a tria (three in a row) or tessera (four in a row). It also keeps track of the number of captures for both players.
